## Key Findings

### search_abbr Function Location
- Defined in: backend/app/services/agent/tools.py (lines 368-437)
- Called by: tool_executor in nodes.py (line 433)
- Database query: Abbreviation.short_form ILIKE "%{abbreviation}%"
- Three return scenarios: not found, single match, multiple matches
- Stored in state: abbreviation_results list[dict]

### RAG & Writing Nodes Interaction
- RAG path: agent_rag_executor (line 1014) → invokes agent_rag subgraph
  * Handles: search, list_docs, summarize, kg_query, search_doc_num, search_abbr
  * Returns: sources, images, kg_summaries, abbreviation_results
  * Can loop back if abbreviation expansion detected
  
- Write path: write_executor (line 877) → invokes agent_write subgraph
  * Handles: write_summarize, write_suggest_edits, write_grammar_check
  * State transform: AgentState → AgentWriteState
  * Output transform: AgentWriteState → AgentState

- Both subgraphs are lazy-loaded singletons
- State transformation uses _transform_*_input() and _transform_*_output()

### ContextVar Workaround Details
- Problem: LangGraph strips non-TypedDict keys before passing to nodes
- Solution: Python contextvars module
- _event_queue_ctx: ContextVar[asyncio.Queue | None]
- _db_ctx: ContextVar
- nodes call: get_current_db() instead of state.get("_db")
- stream_agent_to_sse() sets context before creating task
- asyncio.create_task() inherits context from parent

## Intent Classification Rules

From _CLASSIFIER_SYSTEM (83 lines of detailed rules):
- greeting: no tool needed
- personal: about user, use memory, no tool
- search: document search needed
- list_docs: list documents needed
- summarize: specific document summary needed
- kg_query: knowledge graph entity lookup
- search_doc_num: search by document number
- search_abbr: ⭐ SPECIAL RULE
  * Extract abbreviation ONLY
  * Remove "là gì?", "có nghĩa là gì?", "là viết tắt của gì?"
  * Preserve exact form (BMNN ≠ BMM)
- write_summarize/suggest_edits/grammar_check: text processing

## Graph Topology

START
  → memory_recall (Graphiti search)
  → intent_classifier (Qwen3-4B)
  → [route by intent]
     ├ greeting/personal → direct_answer → END
     ├ write_* → write_executor → END
     └ search/list/summarize/kg/abbr/doc_num
        → agent_rag_executor
           → [check expanded_query]
              ├ YES + iterations < 3 → loop back to agent_rag_executor
              └ NO → answer_generator → END
